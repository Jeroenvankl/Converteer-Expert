Gratis bestandsconverter — alles blijft in je browser

Hoe werkt het?

  • 📁 Je bestanden worden nooit geüpload — alles wordt lokaal in je browser verwerkt
  • 🖼️ Afbeeldingen: JPEG, PNG, WebP, BMP, GIF, HEIC, SVG, TIFF → alle richtingen
  • 🎬 Video: MP4, AVI, WebM, MOV, MKV → inclusief audio-extractie en GIF
  • 🎵 Audio: MP3, WAV, OGG, FLAC, AAC, M4A → alle richtingen
  • 📄 Documenten: PDF, DOCX, Markdown, CSV, JSON → diverse formaten
  • ⚠️ Video/audio-conversies laden eenmalig FFmpeg (~30 MB)
  • 🎬 YouTube: Plak een link → download als MP3 of MP4 (vereist lokale server + yt-dlp)
